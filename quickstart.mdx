title: "Quickstart"

---

This guide walks you through running `kbot` for the first time. By the end, you'll have the bot join a live Kahoot game and answer every question automatically.

## Prerequisites

- Python 3.9 installed ([Installation guide](/installation))
- Dependencies installed: `python3.9 -m pip install -r requirements.txt`
- A live Kahoot game in progress (you need the game pin)
- The quiz's UUID **or** a Kahoot account to search by name

<Steps>
  <Step title="Get the game pin">
    The game pin is the 6–7 digit code shown on the Kahoot lobby screen when the host starts the game. You'll need this to join.

    Example: `123456`
  </Step>

  <Step title="Get the quiz ID">
    The quiz ID is a UUID that uniquely identifies the Kahoot. It looks like:

    ```
    xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
    ```

    You can find it in the URL when you open the quiz on [create.kahoot.it](https://create.kahoot.it):

    ```
    https://create.kahoot.it/details/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
    ```

    <Tip>
      If you don't have the quiz ID, you can search by name instead — but this requires a Kahoot account. See [Authentication](/usage/authentication) for details.
    </Tip>
  </Step>

  <Step title="Run the bot">
    Open a terminal, navigate to the directory containing `kbot`, then run:

    ```bash
    python kbot -n "MyNickname" -p 123456 -i "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
    ```

    Replace:
    - `MyNickname` — the nickname the bot will join with
    - `123456` — the game pin shown on the host's screen
    - `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` — the quiz UUID

    If you want to search by name instead of ID (requires login):

    ```bash
    python kbot -n "MyNickname" -p 123456 -q "My Quiz Title" -e user@example.com -a mypassword
    ```
  </Step>

  <Step title="Watch it answer">
    Once connected, you'll see output like:

    ```
    ANSWERS RECEIVED
    GET READY
    ------ 1 ------
    SELECTED BLUE
    TIME UP
    ------ 2 ------
    SELECTED RED
    TIME UP
    ------ 3 ------
    SELECTED GREEN
    GAME OVER
    ```

    The bot automatically submits the correct answer color for each question. When the game ends, it exits cleanly.

    <Note>
      If the host enabled **2FA**, you'll be prompted to enter the color sequence shown on screen (e.g. `rbyg` for Red, Blue, Yellow, Green). See [Authentication](/usage/authentication) for details.
    </Note>
  </Step>
</Steps>

## Common invocations

<CodeGroup>

```bash By quiz ID (no login required)
python kbot -n "BotName" -p 123456 -i "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
```

```bash By quiz name (login required)
python kbot -n "BotName" -p 123456 -q "Quiz Title" -e user@example.com -a mypassword
```

```bash Interactive mode (prompted for all inputs)
python kbot
```

```bash Preview answers without joining
python kbot -s -i "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
```

</CodeGroup>
