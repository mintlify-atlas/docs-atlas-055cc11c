## What is Kahoot Answer Bot?

Kahoot Answer Bot (`kbot`) is a Python command-line tool that joins a live Kahoot game and automatically submits the correct answer for every question. It works by:

1. Looking up the quiz answers from the Kahoot API — either by quiz UUID or by searching with a name
2. Connecting to the Kahoot game server over WebSocket, impersonating a standard browser client
3. Listening for question events and submitting the correct answer index for each question

<Note>
This project was forked from [reteps/kbot](https://github.com/reteps/kbot) and updated with 2FA support, search-by-ID, and various fixes to work with the current Kahoot API.
</Note>

## Caveats

<Warning>
Kahoot Answer Bot does **not** work when the host has enabled answer randomization or question randomization. It relies on the original answer order from the API, so shuffled answers will produce incorrect selections. See [Caveats](/reference/caveats) for details.
</Warning>
