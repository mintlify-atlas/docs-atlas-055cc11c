Kahoot Answer Bot (`kbot`) is a Python command-line tool that joins a live Kahoot game and automatically submits the correct answer for every question — giving you a perfect score without lifting a finger.

## How it works

`kbot` operates in three stages:

1. **Answer lookup** — fetches the quiz answers from the Kahoot API using the quiz UUID or a name search
2. **Game join** — connects to the Kahoot game server over WebSocket, impersonating a standard browser client, and joins using your chosen nickname and the game pin
3. **Auto-answer** — listens for question events and submits the correct answer index for each question in real time

<Note>
This project was forked from [reteps/kbot](https://github.com/reteps/kbot) and extended with 2FA support, search-by-ID, and various fixes to work with the current Kahoot API.
</Note>

## Key features

<CardGroup cols={2}>
  <Card title="Search by name or ID" icon="magnifying-glass">
    Find any public Kahoot by its UUID or by searching for it by name. Name search requires a Kahoot account.
  </Card>
  <Card title="2FA support" icon="shield-check">
    Handles Kahoot's two-factor authentication challenge automatically during game join.
  </Card>
  <Card title="Search-only mode" icon="eye">
    Preview quiz answers without joining a game using the `-s` flag. Useful for inspecting a quiz before playing.
  </Card>
  <Card title="Debug mode" icon="bug">
    Enable verbose output with `-d` to inspect raw API responses and WebSocket messages during execution.
  </Card>
</CardGroup>
